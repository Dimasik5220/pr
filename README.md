Анализ движения объектов в видеопотоке методами оптического потока
Данный проект предназначен для анализа видеопотока и автоматического обнаружения движущихся объектов в реальном времени. Программа определяет количество движущихся объектов, отслеживает их траектории, определяет направление движения, подсчитывает уникальные объекты за всё время просмотра и сохраняет результаты анализа.

Проект использует компьютерное зрение на базе OpenCV для анализа видеоданных и реализует классические методы оптического потока: Lucas-Kanade (разреженный) и Farneback (плотный), а также методы детектирования движения: absdiff и MOG2.

Что делает программа
Программа автоматически:

загружает видео из папки video

обнаруживает движущиеся объекты в кадре

отслеживает траектории движения объектов (зелёные линии)

определяет направление движения (вправо/влево/вверх/вниз)

подсчитывает количество объектов в текущем кадре

подсчитывает уникальные объекты за всё время просмотра видео

определяет максимальное количество объектов одновременно

выделяет объекты ограничивающими прямоугольниками

показывает центр каждого объекта

отображает ID объекта

показывает цветовую карту плотного оптического потока

сохраняет скриншоты результатов

выводит итоговую статистику в консоль

Методы анализа движения
1. Lucas-Kanade (разреженный оптический поток)
Отслеживает характерные точки (углы) на движущихся объектах

Рисует траектории движения (зелёные линии)

Показывает направление движения (жёлтые стрелки)

Быстрый, подходит для реального времени

2. Farneback (плотный оптический поток)
Вычисляет движение для каждого пикселя кадра

Визуализирует направление и скорость движения цветом

Красный = движение вправо, синий = влево, зелёный = вверх/вниз

Яркость показывает скорость движения

Даёт полную картину движения сцены

3. ABSDIFF (простая разница кадров)
Вычисляет разницу между двумя последовательными кадрами

Простой и быстрый детектор движения

Недостаток: не видит остановившиеся объекты

4. MOG2 (вычитание фона + аналитика)
Строит модель фона на основе истории кадров

Выделяет движущиеся объекты даже при изменении освещения

Подсчитывает уникальные объекты за всё видео

Определяет направление движения

Структура проекта
video/ - папка с видеофайлами. Сюда необходимо загружать видео для обработки.

scripts/ - папка с исходными кодами программы.

results/ - папка с результатами обработки. После запуска программы здесь появятся скриншоты.

Поддерживаемые форматы видео
Программа работает со следующими форматами:

.avi

.mp4

.mov

.mkv

Как установить проект
Скачайте репозиторий.

Убедитесь, что у вас установлен Python версии 3.7 или выше.

Установите необходимые библиотеки:

bash
pip install opencv-python numpy
Как пользоваться
Поместите видеофайл для анализа в папку video.

Запустите программу:

bash
python main.py
После запуска программа автоматически найдёт первое видео в папке и покажет меню выбора режима:

text
============================================================
ВЫБЕРИ РЕЖИМ:
============================================================
 1. Lucas-Kanade (траектории объектов)
 2. Farneback (плотный поток, цвет)
 3. ABSDIFF (простая разница кадров)
 4. MOG2 + подсчёт уникальных объектов за всё видео
 5. Сравнить FPS
 6. Запустить всё подряд
 0. Выход
============================================================
Для каждого режима программа:

обрабатывает видеопоток в реальном времени

отображает движущиеся объекты

показывает траектории и направление движения

считает количество объектов

Управление во время работы:

ESC - выход из текущего режима

s - сохранить скриншот текущего кадра

Что отображается на экране
Режим Lucas-Kanade:
Зелёная линия - траектория движения объекта

Красная точка - текущее положение объекта

Жёлтая стрелка - направление движения

Белая цифра - ID объекта

Счётчик объектов в левом верхнем углу

Режим Farneback:
Верхняя половина - исходное видео

Нижняя половина - цветовая карта движения

Красный цвет = движение вправо

Синий цвет = движение влево

Зелёный цвет = движение вверх/вниз

Яркость = скорость движения

Режим ABSDIFF:
Верхняя часть - исходное видео

Средняя часть - маска движения (бинарное изображение)

Нижняя часть - обнаруженные объекты с прямоугольниками

Режим MOG2:
Зелёный прямоугольник - граница объекта

Жёлтая линия - траектория движения

Синяя стрелка - направление движения

Белая цифра - ID объекта

Текущее количество объектов в кадре

Всего уникальных объектов за всё видео

Максимум объектов одновременно

Итоговая статистика
После завершения обработки видео в режиме MOG2 программа выводит в консоль:

text
============================================================
ИТОГОВАЯ СТАТИСТИКА ПО ВИДЕО:
============================================================
  Всего уникальных объектов за видео: 15
  Максимальное количество одновременно: 3
  Всего обработано кадров: 300
============================================================
Сравнение методов
Метод	Скорость (FPS)	Детальность	Применение
Lucas-Kanade	15–30	Низкая (центры объектов)	Трекинг, подсчёт объектов
Farneback	2–5	Высокая (каждый пиксель)	Анализ сложного движения
ABSDIFF	30–60	Низкая (маска движения)	Быстрое детектирование
MOG2	10–20	Средняя (объекты + траектории)	Подсчёт и аналитика
Возможные ошибки
Ошибка: В папке video нет видео файлов
Причина: В папке video нет видеофайлов.

Решение: Добавьте файлы .avi, .mp4, .mov или .mkv в папку video.

Ошибка: Не удалось открыть видео
Причина: Файл повреждён или кодек не поддерживается.

Решение: Проверьте видеофайл, попробуйте другой формат.

Ошибка: OpenCV не установлена
Причина: Не установлена библиотека OpenCV.

Решение: Выполните команду:

bash
pip install opencv-python
Особенности работы
все видео автоматически обрабатываются покадрово

очень маленькие объекты (площадь менее 400 пикселей) игнорируются

программа автоматически выбирает первое видео из папки video

для остановки обработки можно нажать клавишу ESC

в режиме MOG2 ведётся подсчёт уникальных объектов за всё время
